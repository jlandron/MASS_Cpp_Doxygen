◆ getCores()

int MASS_base::getCores ( )
static

Returns the number of cores currently available on the machine. If -1 is returned, then the option is not supported on this platform. If zero is returned, then it means that the relevant functions and headers exist, but one has to ask at run time what degree of support is available. Else, will return the number of available cores.

Returns
numeric representation of the number of cores available on this machine (will not count configured cores that are offline/unavailable at runtime)

◆ log()

void MASS_base::log ( string  msg)
static

Logs a message to Mass's internal logs. If this instance of MASS represents the master node (myPid == 0), then logs are printed directly to the cerr output. Otherwise, all slave nodes print out messages to the following file:

<CUR_DIR>/MASS_logs/PID_<myPid>_<hostname>result.txt

Parameters
msgString of message to print to output log
