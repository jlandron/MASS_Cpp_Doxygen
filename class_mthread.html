◆ run()

void * Mthread::run ( void *  threadId)
static

This method begins the run process for a threaded task in MASS (Mthread). Effectively, it starts a loop where the Mthread evaluates the next task to perform, performs it, and then waits for other threads to complete before moving on to the next task.

Parameters
threadIda reference to the address that contains the thread ID for the actual Mthread instance to run
Returns
currently returns NULL when complete, can/should be adapted to pass through (return) values associated with relevant call (e.g. - callAll())
